# 모두의 게시판
## 요구사항
### 게시판
- [ ] 글 목록을 확인할 수 있다.
- [ ] 글을 작성할 수 있다.
  - [ ] 댓글을 작성할 수 있다.
  - [ ] 댓글을 수정할 수 있다.
  - [ ] 댓글을 삭제할 수 있다.
- [ ] 글을 수정할 수 있다.
- [ ] 글을 삭제할 수 있다.
- [ ] 게시글을 검색할 수 있다.
### 사용자
- [ ] 회원가입을 할 수 있다.
  - [ ] 회원가입 시 이메일(인증), 아이디, 비밀번호, 이름을 받는다.
- [ ] 로그인을 할 수 있다.
- [ ] 회원 정보를 확인할 수 있다.
- [ ] 아이디를 찾을 수 있다.(이메일 인증 후)
- [ ] 비밀번호를 찾을 수 있다.(비밀번호 찾기)
- [ ] 비밀번호를 변경할 수 있다.
- [ ] 사용자가 작성한 글을 볼 수 있다.
- [ ] 시용자가 작성한 댓글을 볼 수 있다.
